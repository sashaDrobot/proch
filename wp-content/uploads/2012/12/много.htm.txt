<p class=MsoNormal><span class=hps>Разложить</span> <span class=hps>многочлен</span>
<span class=hps>на множители</span> <span class=hps>означает превратить</span> <span
class=hps>многочлен</span> <span class=hps>на произведение</span> <span
class=hps>двух или нескольких</span> <span class=hps>многочленов.</span> <span
class=hps>Есть четыре способа</span> <span class=hps>разложения</span> <span
class=hps>многочлена</span> <span class=hps>на множители.</span><br>
<br>
<span class=hps>Один из них -</span> <span class=hps>это вынесение</span> <span
class=hps>общего множителя</span> <span class=hps>за скобки.</span><br>
<br>
<span class=GramE><span class=hps>Чтобы вынести</span> <span class=hps>общий
множитель</span> <span class=hps>за скобки</span>, <span class=hps>надо:</span><br>
<br>
<span class=hps>1)</span> <span class=hps>найти общий</span> <span class=hps>множитель;</span><br>
<br>
<span class=hps>2)</span> <span class=hps>разделить на</span> <span class=hps>него
каждый</span> <span class=hps>член</span> <span class=hps>многочлена</span> <span
class=hps>и полученную сумму</span> <span class=hps>взять в</span> <span
class=hps>скобки;</span><br>
<br>
<span class=hps>3)</span> <span class=hps>записать</span> <span class=hps>произведение</span>
<span class=hps>общего множителя</span> <span class=hps>на</span> <span
class=hps>полученную сумму.</span></span><br>
<br>
<br>
<span class=hps>Примечание:</span> <span class=hps>если при вынесении</span> <span
class=hps>за скобки</span> <span class=hps>общий множитель</span> <span
class=hps>выносится</span> <span class=hps>со знаком</span> <span class=hps>«-»</span>,
<span class=hps>то знаки</span> <span class=hps>слагаемых в</span> <span
class=hps>скобках</span> <span class=hps>меняются <span class=GramE>на</span>
противоположные.<o:p></o:p></span></p>

<p class=MsoNormal><span class=hps><o:p>&nbsp;</o:p></span></p>

<p><span class=hps>Пример 1.</span><span class=shorttext> </span><span
class=hps>Разложить на множители</span><span class=shorttext> </span><span
class=hps>многочлен</span><span style='mso-spacerun:yes'>  </span>12х<sup>2</sup>у-18х<sup>2</sup>у<sup>2</sup></p>

<p><span class=hps>Сначала</span> <span class=hps>найдем общий</span> <span
class=hps>числовой множитель</span> <span class=hps>для коэффициентов</span> <span
class=hps>12 и</span> <span class=hps>-18,</span> <span class=hps>если</span> <span
class=hps>коэффициентами</span> <span class=hps>являются целые числа</span>, <span
class=hps>то</span> <span class=hps>за общий</span> <span class=hps>числовой
множитель</span> <span class=hps>берут, как</span> <span class=hps>правило</span>,
<span class=hps>наибольший общий делитель</span> <span class=hps>модулей</span>
<span class=hps>этих коэффициентов.</span> <span class=hps>В нашем</span> <span
class=hps>случае - это</span> <span class=hps>число 6.</span> <span class=hps>Степени</span>
<span class=hps>с основанием</span> <span class=SpellE><span class=hps>х</span></span>
<span class=hps>входят</span> <span class=hps>в</span> <span class=hps>оба
члена</span> <span class=hps>многочлена.</span> <span class=hps>Поскольку
первый</span> <span class=hps>член</span> <span class=hps>содержит</span> х<sup>3</sup>=х<span
class=GramE><sup>2</sup></span> <span class=SpellE>х</span>, а другой<span
style='mso-spacerun:yes'>  </span>- х<sup>2</sup>, <span class=hps>то</span><span
class=shorttext> </span><span class=hps>общий множителем</span><span
class=shorttext> </span><span class=hps>для степеней</span><span
class=shorttext> </span><span class=hps>с основанием</span> <span class=SpellE>х</span>
<span class=SpellE>є</span> х<sup>2</sup> <span class=hpsatn>(</span>за скобки <span
class=hps>выносят</span> <span class=hps>переменную</span> <span class=hps>с
меньшим показателем</span>). <span class=hps>В члены</span> <span class=hps>многочлена</span>
<span class=hps>входят соответственно</span> <span class=hps>множители</span> у
<span class=SpellE>і</span> у<span class=GramE><sup>2</sup></span>, за скобки
можно вынести у. Таким образом, за скобки можно вынести<span
style='mso-spacerun:yes'>  </span>6х<sup>2</sup>у: </p>

<p>12х<sup>2</sup>у-18х<sup>2</sup>у<sup>2</sup>=6х<sup>2</sup>у 2х-6х<sup>2</sup>у
3у=6х<sup>2</sup>у(2х-3у). </p>

<p><span class=hps>Пример.</span><span class=shorttext> </span><span class=hps>Разложить</span><span
class=shorttext> </span><span class=hps>на</span><span class=shorttext> </span><span
class=hps>множители</span><span class=shorttext> </span><span class=hps>многочлен</span>
5<span class=GramE>х(</span><span class=SpellE>а-с</span>)+3(<span
class=SpellE>а-с</span>) </p>

<p><span class=hps>Данное выражение</span> <span class=hps>является суммой двух</span>
<span class=hps>слагаемых</span>, <span class=hps>для которых</span> <span
class=hps>общим</span> <span class=hps>множителем</span> <span class=hps>есть
выражение</span> <span class=SpellE><span class=hps>а</span>-с</span>. <span
class=hps>Вынесем этот</span> <span class=hps>множитель за</span> <span
class=hps>скобки:</span>5<span class=GramE>х(</span><span class=SpellE>а-с</span>)+3(<span
class=SpellE>а-с</span>)=(<span class=SpellE>а-с</span>)(5х+3) </p>

<p><span class=hps>Разложить</span><span class=shorttext> </span><span
class=hps>на</span><span class=shorttext> </span><span class=hps>множители</span><span
class=shorttext> </span><span class=hps>многочлен </span><span
style='mso-spacerun:yes'> </span>2<span class=GramE>х(</span><span
class=SpellE>а-с</span>)+у(<span class=SpellE>с-а</span>) </p>

<p><span class=hps>Слагаемые</span> <span class=hps>имеют</span> <span
class=hps>множители</span> <span class=SpellE><span class=hps>а-с</span></span>
<span class=hpsatn>и <span class=SpellE><span class=GramE>с-а</span></span></span>,
<span class=hps>которые отличаются только</span> <span class=hps>знаками.</span>
<span class=hps>В выражении</span> <span class=SpellE><span class=GramE><span
class=hpsatn>с-</span>а</span></span> <span class=hps>вынесем за скобки</span> <span
class=hps>-1,</span> <span class=hps>тогда</span> <span class=hps>второе
слагаемое</span> <span class=hps>будет иметь <span class=SpellE>вид<span
class=atn>-у</span></span></span><span class=atn> (</span><span class=SpellE>а-с</span>)
<span class=hps>и оба</span> <span class=hps>слагаемые</span> <span class=hps>будут</span>
<span class=hps>общий множитель</span> <span class=SpellE><span class=hps>а</span>-с</span>.
Таким образом, 2<span class=GramE>х(</span><span class=SpellE>а-с</span>)+у(<span
class=SpellE>с-а</span>)= 2х(<span class=SpellE>а-с</span>)-у(<span
class=SpellE>а-с</span>)=(<span class=SpellE>а-с</span>)(2х-у) </p>

<p>Приклад 5<b>.</b> <span lang=UK style='mso-ansi-language:UK'>Найти </span>значение
выражения <span style='mso-ansi-language:UK'><span
style='mso-spacerun:yes'> </span></span><span
style='mso-spacerun:yes'> </span>8,5а<sup>2</sup>+а<sup>3</sup>, если а=1,5 </p>

<p>Разложим сначала <span style='mso-spacerun:yes'> </span>8,5а<span
class=GramE><sup>2</sup></span>+а<sup>3</sup> <span class=hps>на</span><span
class=shorttext> </span><span class=hps>множители</span> </p>

<p>8,5а<span class=GramE><sup>2</sup></span>+а<sup>3</sup>.=а<sup>2</sup>(8,5+а)
</p>

<p>Если<span style='mso-spacerun:yes'>  </span>а=1,5, то<span class=GramE> :</span>
</p>

<p>а<span class=GramE><sup>2</sup></span>(8,5+а)=1,5<sup>2</sup>(8,5+1,5)=2,25*10=22,5
</p>
